 HR Analytics Dashboard - Employee Attrition Analysis

 Tools Used
- Power BI
- DAX
- Data Visualization

 Project Overview
This project analyzes employee attrition trends using Power BI. The dashboard provides insights into employee turnover, age distribution, education fields, job roles, and overtime impact.

 Key Metrics
- Total Employees
- Attrition Count
- Attrition Rate
- Average Age
- Average Tenure
- Average Monthly Income

Dashboard Insights
- Attrition by Department
- Attrition by Age Group
- Attrition by Education Field
- Attrition by Job Role
- Attrition by Years at Company

Dataset:
The dataset used in this project is a public HR Analytics dataset commonly used for Power BI practice and learning purposes.
